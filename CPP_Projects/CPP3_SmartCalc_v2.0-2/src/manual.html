Проект SmartCalc v2.0

Проект разрабатывался на ubuntu 22.04
Цели в Makefile

- make all установка калькулятора и запуск тестов
- make install установка калькулятора
- make run запуск калькулятора
- make unistall удаление калькулятор
- make clean очистка неиспользуемых файлов
- make dvi открытие manual по калькулятору
- make dist создание архива проекта
- make test запуск unit тестов
- make gcov_report запуск unit тестов и отображение покрытия кода тестами
